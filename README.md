💻 ЗАДАНИЕ № 9:

Создание простой веб-страницы

Чтобы потренироваться в понимании HTML и CSS, создадим ваше первое портфолио со следующей структурой:

1. В названии вкладки в браузере должно фигурировать ваше имя и фамилия.
2. Дизайн, стиль, оформление вы выбираете сами и включаете его в файл styles.css.
3. Ресурсы по работе с HTML и CSS: http://htmlbook.ru и https://www.w3schools.com/html
4. Созданную страницу вы можете загрузить в браузере, используя плагин Live Server в Visual Studio Code или контекстное меню "Open in default browser"

В качестве ответа прикрепите ссылку на гугл-диск со скриншотом (вместе с окном браузера и адресной строкой). На скриншоте должны быть видны ваши URL, название вкладки и содержимое веб-страницы.
